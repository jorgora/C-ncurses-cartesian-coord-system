#+title: Readme

* C - cartesian coordinate system.
They say programming is about using the right tool for the right job so I made a cartesian coordinate system using ncurses and C.

A program that can display math functions such as sin(x) or x^2 would be both useful for programming and good for a portfolio.
Ncurses has a resolution of roughly 100 * 200 characters full screen on my terminal so it's not the most precise toold for the job, but it's retro.

This program was intended mostly for art. An ncurses sine wave is funny. It draw on points of (x,y) coords,  or draw rects in order to make retro art.
